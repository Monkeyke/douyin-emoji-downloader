# 抖音收藏表情包批量下载器 (Douyin Emoji Downloader)

这是一个基于 Python 和 Playwright 的半自动化脚本，用于批量下载你在抖音网页版中收藏的自定义表情包。

## ✨ 特性

- **半自动化操作**：脚本会自动打开浏览器，允许你手动扫码登录并定位到表情面板，避免了复杂的登录验证码和反爬虫机制。
- **精准提取**：通过分析图片 URL 特征（包含 `emoticon` 或 `emotion`），100% 精准锁定表情包，不会误抓头像或普通图片。
- **穿透 iframe**：自动遍历页面中的所有框架，即使聊天窗口嵌套在 iframe 中也能成功提取。
- **自动去重与格式识别**：自动过滤重复的表情，并根据 URL 或 Content-Type 智能识别文件格式（如 `.webp`, `.gif`, `.png` 等）。

## 🛠️ 环境准备

1. 确保你的电脑上安装了 **Python 3.8+**。
2. 克隆或下载本仓库代码到本地。
3. 安装依赖包：

```bash
pip install -r requirements.txt
```

4. 安装 Playwright 浏览器内核（如果脚本无法调用系统自带的 Chrome/Edge，可能需要执行此步）：

```bash
playwright install chromium
```

## 🚀 使用指南

1. 在终端中运行脚本：

```bash
python douyin_emoji_downloader.py
```

2. 脚本会自动启动一个 Chrome（或 Edge）浏览器窗口，并打开抖音首页。
3. **请在浏览器中依次完成以下手动操作**：
   - 扫码登录你的抖音账号。
   - 点击左侧导航栏的 **「消息」**，进入任意一个私信聊天窗口。
   - 点击输入框旁边的 **「😊」表情按钮**。
   - 切换到 **「自定义表情」**（即你收藏的表情）标签页。
   - **重要：** 手动在表情面板中**向下滚动**，确保所有你想下载的表情图片都已经加载显示出来。
4. 操作完成后，回到运行脚本的终端窗口，按下 **`Enter` (回车键)**。
5. 脚本将自动扫描页面，提取所有表情包的链接，并下载到当前目录下的 `抖音收藏表情包` 文件夹中。

## ⚠️ 注意事项

- 脚本依赖于浏览器中实际渲染的 DOM 元素，因此**必须手动滚动表情面板**，让图片加载到页面中，脚本才能抓取到。
- 下载的图片格式通常为 `.webp` 或 `.gif`，这是抖音服务器存储的原始格式。
- 请合理使用，遵守相关平台的规定。
